## Amazon Clone

A fully interactive Amazon clone built using HTML, CSS, and JavaScript. This project replicates the layout and basic functionality of Amazon’s online store, including product listings, add-to-cart interactions, and responsive design for desktop and mobile screens.

Features

- Interactive product listing with images, prices, and descriptions.

- Add to cart functionality using JavaScript.

- Responsive design for mobile, tablet, and desktop screens.

- Fully built with HTML, CSS, and vanilla JavaScript — no frameworks.

- Provides hands-on experience with DOM manipulation, events, and styling.

  This project is live at:
https://heyayanesh.github.io/Amazon-clone/amazon.html

Tech Stack

- HTML for structure

- CSS for styling and responsive design

- JavaScript for interaction and dynamic features
